学校行为规范目标、规划要点、保障机制

行为规范目标

　　进一步光大学生“品行高尚、学习勤奋、兴趣广泛、思维敏捷、学业精良、善于处事”的良好形象
　　进一步弘扬学生“追求真理、探究真知、崇尚真情、塑造真我”的求真精神。

 　

 
   基于对集体、民族和国家责任感、使命感的爱国奉献精神。
   基于对全球化、国际化、网络化人类发展趋势认识的，体现国际视野和开放意识的全球精神。
   基于对自然与社会及其相互关系认识的可持续发展精神。
   基于对科技飞速发展趋势的认识，形成终身学习的意识和能力，有科学探究、科技应用和科学理解能力，有艰苦奋斗的创业创新精神。
   基于适应未来社会复杂多变的发展趋势，有健全的人格和强健的体魄，具有竞争和合作精神。

规划要点: 
     1．“复兴人”教育的目标化、循序化

        活动之一：“新世纪‘复兴人’形象”探讨
       活动之二：“ 新世纪‘复兴人’精神”探究
       活动之三：“我诚信，我快乐”
       活动之四：“我所认识的网络文化”
       活动之五：“世博，复兴人为你添彩”
       活动之六：“世博，我为你添彩，迈好人生每一步”
  　活动之七：“弘杨中华民族精神，培育上海城市精神－争做新世纪复兴人”
    活动之八：“纪念抗日战争胜利60周年和世界反法西斯战争胜利60周年”

　

 2．班级建设的规范化、个性化

   3．学生团体的创新化、自主化（以团学联创立的学生活动中心，体现学生参与学校管理，开展自主学习、自主教育、自主管理的机制）

       

4．社会实践的社会化、体验化（高一军训与高二学农、党课理论小组活动、寒暑假社会考察调查、社会志愿者服务队、校园劳动制度五个方面的落实）  

      5．心理健康的有序化、多样化（开展心理调查、心理辅导、心理咨询，组织形式多样的活动

　

　

保障机制:

1．管理机构建全
德育校长下设两个机构：政教-年级组-班级和团学联-年级团总支-班级团支部，开展行为规范的教育与管理。  

　

  

2．评比机制完善
   学校依据校内行为规范条例每周进行各班评比（班级、年级、校级
三级评比）；内容包括：准时到校、两微佩戴、升旗两操、三关及时、班级卫生、公益劳动、校会纪律、自修认真、文明礼仪、宿室管理等十个项目。评出每月的优胜班，颁发流动红旗，为各年级日常行为规范评比起到良好的示范作用。
   学校团总支组织各班团支部开展值周评比；
   年级组组织学生参与年级行为规范评比；
   学校进行班主任工作细化指标实施细则。

3．师德典范发扬
 学校大力宣传发扬爱岗敬业、教书育人、为人师表、热爱学生的先进思想和事迹，提倡班主任教育管理工作的创新精神

4．家校合力作用
 学校组织家委会、定期召开家长会、开展家访及电话交流等多渠道与家长沟通联系。

5．科研工作促进
学校加强科研工作，涉及到国家级、市级、区级等多项课题，促进德育工作。
